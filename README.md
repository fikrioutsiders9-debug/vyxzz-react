## 🛠️ Advanced Tech Stack & Architecture
- **Frontend Core:** React.js (Functional Components & Custom Hooks)
- **Build Pipeline:** Vite (Advanced bundling & tree-shaking)
- **State Management:** Zustand (Lightweight, centralized atomic state to prevent redundant re-renders)
- **Routing Engine:** React Router v6 (Implementing `useParams` for scalable dynamic service-detail injection)
- **Database Backend:** Supabase (Real-time data streaming & secure cloud architecture)
- **Deployment Platform:** Vercel

---

## 🏗️ Key Engineering Features
- **Dynamic Content Architecture:** Fully decoupled frontend that dynamically renders complex service detail views based on secure URL parameters (`useParams`).
- **Centralized State Layer:** Managed global UI states and asynchronous database filters globally via **Zustand** without prop-drilling.
- **Media Asset Optimization:** 100% migrated to compressed next-gen `.webp` formatting, slashing initial load times and boosting performance score to **98**.
- **Responsive Layout Engine:** Handcrafted, fluid CSS grid/flexbox architecture designed for flawless mobile & desktop fidelity.

# Vyxzz Portfolio - React Version

This project is a modern re-build of my first portfolio website, migrated from Vanilla JavaScript to **React.js** with **Vite**.

## 🚀 Tech Stack
- **Library:** React.js
- **Build Tool:** Vite
- **Styling:** CSS3 (with Responsive Design)
- **Icons:** Font Awesome & Custom SVGs
- **Deployment:** Vercel

## 🛠️ Features
- **Modern Workflow:** Built using functional components and React Hooks (`useState`).
- **Responsive Navigation:** Integrated Navbar and Sidebar logic using state management.
- **Optimized Performance:** Fast loading times thanks to Vite's build system.

## 🏗️ Setup & Installation
1. Clone the repository:
   `git clone https://github.com/fikrioutsiders9-debug/vyxzz-react.git`
2. Install dependencies:
   `npm install`
3. Run development server:
   `npm run dev`
